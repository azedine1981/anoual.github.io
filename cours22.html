Séquence 2: Système d'exploitation réseau
1- Notion de système d'exploitation réseau

Le système d’exploitation gère l’allocation et l’utilisation de toutes les ressources de l’ordinateur, et coordonne les interactions entre l’utilisateur et les programmes qui sont exécutés sur l’ordinateur. Un système d’exploitation réseau permet ainsi :

  • Le contrôle de l’accès des utilisateurs au réseau ;
  • Le partage des ressources ;
  • La surveillance du réseau.

La gestion et l’administration des réseaux nécessitent un système d’exploitation spécialisé comme Linux, Windows 2016 server…

2- Gestion des utilisateurs d’un réseau local
A- Créer un compte d'utilisateur

Un compte d’utilisateur est un profil unique qui inclut tous les paramètres privilégiés et fichiers personnels de chaque utilisateur. Les comptes Windows permettent à l’administrateur de contrôler l’accès de chaque utilisateur.
Pour créer un compte sous Windows 10Windows, on procède comme suit :
• connecte-toi à ton ordinateur en tant qu’administrateur ;
• Clique sur Démarrer/Panneau de configuration/compte d’utilisateur ;
• Clique sur Créer un nouveau compte ;
• Saisis le nom de l’utilisateur ;
• Clique sur Suivant ;
• Pour le type de compte, choisis Utilisateur , Puis sur Créer un compte ;
• Le nouveau compte crée paraît dans l’écran de bienvenue et « Gérer les comptes ».
(Dans la suite de notre cours l’étude se restreint sur les réseaux sans fil WIFI « SSID »)

B- Se connecter à un point d'accès

Dans la zone des notifications de la barre des tâches, clique sur l’icône réseau puis choisis le nom du point d’accès désiré (saisis le mot de passe s’il s’agit d’une première connexion).
