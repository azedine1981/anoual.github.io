Des exemples de contrôles surveillés précédents


Exemple 1:(2023-2024)

1- Un réseau informatique est un réseau de:
| | Villes;
| | Lignes téléphoniques;
| | Ordinateurs.
2- Donne deux avantages des réseaux informatiques.
3- Comment s'appelle un réseau crée par des câbles?
4- complète le tableau en-dessous par les éléments suivants:
Adaptateur WiFi; Carte réseau; Switch; Hub; Point d'accès; Câble RJ45; Cäble à fibres optiques.
Réseau filaire Réseau sans fil
... ...
5- Que signifie le mot "topologie des réseaux"?
| | Comment communiquent les ordinateurs;
| | La sécurité des ordinateurs;
| | La distribution des réseau dans un réseau.
6- Shématise la topologie en anneau (en donnant les éléments essentiels).
7- Donne la configuration matérielle d'un réseau filaire de 20 ordinateurs.
8- Donne la configuration matérielle d'un réseau sans fil de 6 ordinateurs.
9- si on débranche un ordinateur tout le réseau tombe en panne dans la topologie:
| | En bus; | | En étoile; | | En anneau.
