Séquence1: Système informatique • l’information est un ensemble de données ayant un sens. • l’information peut se présenter sous différentes formes : texte, image, son, vidéo… • Le traitement est une suite d’opérations effectuées pour passer d’un état initial à un état final (recherché, amélioré, désiré). • on distingue trois types de traitements différents : • Manuellement ; • Semi-automatique ; Automatique.
